# Inbox

Raw capture surface. Append one bullet per thought from Obsidian mobile — no
structure needed at capture time. A session clears this with the
`inbox-triage` skill, which sorts each item and asks you to confirm where it
goes.

Drop standalone `.md` notes in this folder too (mobile "new note"); triage
treats each as one item.
